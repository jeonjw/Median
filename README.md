
# 소개

아주대학교 미디어학과를 위한 Android Application 입니다.
모바일 프로그래밍 수업때 프로젝트로 진행했고 기능은 
학과 공지사항, 게시판, 학과 시간표, 학과 정보의 기능을 가지고있고
학과 생활에 편의성을 돕기위해 공지 푸시알림, 학부 사무실 전화걸기 등 편리하고 유용한 기능을 담으려고 했습니다.

개발 : 전진우, 이도원
