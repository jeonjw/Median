Funkey는 아주대학교 미디어학과 미디어프로젝트 과목에서 진행된 아동 운동 보조 어플리케이션입니다. 블루투스 하드웨어로부터 운동신호를 받아와서 운동의 성공여부를 판단하며 보다 운동을 재미있게 할 수 있도록 아이의 운동을 보조해 줍니다.  
Android 개발 : 전진우, 심민제  
디자인 : 정재웅
